# 不可变数据结构 
## 是啥？ 有啥用？ 解决了啥问题？



1. 可变的数据结构带来的问题
```js
var obj = { a: 1 };
var copyObj = obj;
copyObj.a = 2;
console.log(obj.a); // 2
```

2. 避免问题的思路 
- deepCopy 原数据结构。
- 返回修改过的deepCopy
可能存在的缺点：如何处理深层嵌套的对象？

3. immutable.js 解决的问题
immutable.js 是一个提供多种不可变数据结构的库。不可变数据结构的特点在于对于其的引用可更改，但是其内部数据却是不可变的。
这意味着可以使用他们构建可预测的可靠的状态模型。从而方便应用的状态管理。
